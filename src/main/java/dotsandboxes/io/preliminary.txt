PLAYER NAMES:
Default player names are integer numbers.
Custom names can be chosen with the following rules:
  -You are not allowed to choose an integer number as name.
  -Repeated occurrences of the same name are not permitted.
  -If no custom name is given, you will be assigned an increasing integer number.
